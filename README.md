# Penelope AI Assistant

Een 1-op-1 functionele AI assistent die buiten de Cursor GUI werkt. 

## Mogelijkheden
- **Bestandsbeheer**: Lezen, schrijven en oplijsten van bestanden.
- **Terminal Toegang**: Uitvoeren van shell commando's.
- **AI Intelligentie**: Aangedreven door Claude 3.5 Sonnet (of andere modellen).

## Installatie
1. Navigeer naar de projectmap: `cd f:/penelope`
2. Installeer de benodigdheden: `pip install -r requirements.txt`
3. Kopieer `.env.example` naar `.env` en vul je API keys in.
4. Start de assistent: `python run_penelope.py`

## Gebruik
Je kunt direct chatten met Penelope via de command line. Ze heeft toegang tot je bestanden en kan commando's voor je uitvoeren, net zoals in Cursor.
